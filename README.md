# Sabka Bazaar : Shopping Cart

Foobar is a Python library for dealing with word pluralization.

## Installation

Use the package manager [npm](http://npmjs.com/).

```node
npm install
```

## Usage

#### React
```node
npm start
```

#### Backend
```node
npm run server
```
