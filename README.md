# Sabka Bazaar

_Project on E-commerce site._

## Getting Started

**Installation**  
  
Clone the repository and install the dependencies:

```sh
npm i
```

**Development**

> **`Starting both frontend and mock API servers`**

```sh
npm start
```

> **`Starting only frontend server`**

```sh
npm run dev
```

> **`Starting only mock server`**

```sh
npm run server
```

> **`Running test`**

```sh
npm test
```

#### Production

For production release:

```sh
npm run build
```
